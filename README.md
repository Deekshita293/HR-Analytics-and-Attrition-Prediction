# HR Analytics and Attrition Prediction
   A machine learning web app to predict employee attrition and visualize workforce insights.Built with Flask, Bootstrap, and Chart.js for a clean recruiter‑friendly interface.
  
## Features
- Recruiter Sign In / Sign Out flow.
- Prediction Page → enter employee details and get attrition risk (High/Low) with probability.
- Dashboard → charts for attrition risk and job satisfaction levels.
- Professional UI with Bootstrap styling.
- Lightweight, no database required — uses session to store last prediction.
- Powered by a trained Scikit‑learn ML model.

## Installation / How to Run
1.Clone the repository
   git clone https://github.com/Deekshita293/HR-Analytics-and-Attrition-Prediction.git 
   
2.Navigate into the project folder
   cd HR-Analytics-and-Attrition-Prediction
   
3.Install dependencies
   pip install -r requirements.txt
   
4.Run the app
   python app.py
   
5.Open in browser → http://127.0.0.1:5000
